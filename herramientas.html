TORNILLOS DE BANCO

Se compone de una parte fijada en el banco y otra que se mueve mediante un tornillo, entre las que sujeta, apretándola, la pieza que se trabaja.

MANDRILES

Comúnmente usado para sujetar una herramienta rotativa (como las brocas en una taladradora) o en una pieza de trabajo en rotación.

PRENSAS

La prensa es un dispositivo que se utiliza para compactar.Existen distintos tipos de prensa de acuerdo al uso en cuestión.

TROMPOS

El trompo es una máquina herramienta, se usa para fresar y agujerear un área determinada de piezas duras como la madera o el plástico.

HUSILLO

Tornillo metálico o de madera utilizado para el movimiento de las prensas y otras máquinas similares.

PENSA TIPO C

Su función es sujetar la madera que se va a cortar para que no se mueva y se genere un corte desigual.
